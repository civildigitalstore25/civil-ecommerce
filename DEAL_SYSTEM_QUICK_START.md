# Quick Start Guide: Using the Deal System

## For Admins: Creating a Deal

### Step 1: Access Product Form
1. Navigate to Admin Dashboard
2. Click on "Products" tab
3. Either:
   - Click "Add Product" for new product, OR
   - Click "Edit" on existing product

### Step 2: Configure Deal
1. Scroll to "üéâ Deal / Discount Configuration" section
2. Check the box "Enable Deal/Discount Pricing"
3. Set the deal duration:
   - **Deal Start Date**: When the deal should begin (e.g., 2026-02-23)
   - **Deal Start Time**: Time of day to start (e.g., 00:00)
   - **Deal End Date**: When the deal should end (e.g., 2026-02-28)
   - **Deal End Time**: Time of day to end (e.g., 23:59)

### Step 3: Set Deal Prices
The form will show different pricing options based on your product type:

#### For E-books:
- Enter deal prices in INR and USD

#### For Software Products:
- **Subscription Plans**: Enter deal price for each plan (1 Year, 3 Year, etc.)
- **Lifetime Access**: Enter deal price if product has lifetime option
- **Membership**: Enter deal price if product has membership option
- **Admin Subscription Plans**: Enter deal price for each admin plan

### Step 4: Save
1. Click "Add Product" or "Update Product"
2. Confirm the action
3. Deal is now configured!

## How Deals Work

### Automatic Activation
- Deal **automatically activates** at the start date/time you specified
- No manual action needed!

### During Active Deal
- Product detail page shows:
  - ~~Original price~~ (crossed out)
  - **Deal price** (highlighted in gold/yellow)
  - ‚è∞ **Countdown timer** showing time remaining
  - Deal start and end dates
- Product appears in the **üî• Deals** menu
- Customers see deal prices in cart and checkout

### Automatic Deactivation
- Deal **automatically expires** at the end date/time
- Product reverts to regular pricing
- Removed from Deals menu
- No manual intervention required!

## Customer Experience

### Viewing Deals
1. Click **üî• Deals** in the main navigation (desktop or mobile)
2. Browse all active deals in a grid layout
3. Each deal shows:
   - Product image with discount badge (e.g., "20% OFF")
   - Original and deal prices
   - Live countdown timer
   - "View Deal" button

### Purchasing Deals
1. Click on a deal product
2. View full product details with deal pricing
3. Select desired license/subscription type (deal price applied)
4. Add to cart (cart shows deal price)
5. Proceed to checkout (deal price used)

## Tips for Success

### ‚úÖ Best Practices
- Set deal prices **lower** than regular prices (customers expect savings!)
- Use whole hours for start/end times (e.g., 00:00, 23:59) for clarity
- Give deals enough time (minimum 24 hours recommended)
- Test with a short duration first to see how it works
- Monitor the countdown timer to ensure correct timing

### ‚ö†Ô∏è Common Mistakes to Avoid
- Don't set end date before start date (validation will catch this)
- Don't set deal prices higher than regular prices (defeats the purpose)
- Don't rely on manual updates - system is automatic!
- Don't forget to fill in all pricing fields you want to discount

### üí° Pro Tips
- **Weekend Sales**: Friday 00:00 to Sunday 23:59
- **Flash Sales**: Short 6-12 hour deals create urgency
- **Holiday Deals**: Start a few days before, end day after holiday
- **Seasonal Sales**: Month-long deals for clearance
- Combine deal with "Best Seller" badge for maximum impact

## Troubleshooting

### Deal Not Showing?
- Check current date/time is between start and end
- Verify deal checkbox is enabled
- Ensure deal prices are entered correctly
- Refresh the page

### Countdown Timer Not Updating?
- Timer updates every second automatically
- If stuck, refresh the browser
- Check your system clock is correct

### Deal Prices Not Applied?
- Ensure deal is within active dates
- Verify you entered deal prices for the specific plan selected
- Check browser console for errors

### Deal Not in Deals Menu?
- Deal must be active (current time between start/end)
- Product status must be "active" (not draft/inactive)
- Refresh the Deals page
